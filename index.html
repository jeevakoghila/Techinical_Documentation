Data Types

There are various data types in programming, such as int, float, and string.

For example, in Python, you might use type(variable) to check the data type of a variable.

Variable

A variable is a storage location paired with an associated symbolic name. For instance, let x = 10; in JavaScript.

In Python, you would use x = 10.

Control Flow

Control flow in programming refers to the order in which the program's code executes. Examples include if statements and loops like for and while.

Functions

Functions are reusable blocks of code that perform a specific task. They can take inputs, process them, and return an output.

Example in Python:

def greet(name):
    return f"Hello, {name}!"
Modules

Modules are files containing Python definitions and statements. They help in organizing and reusing code.

To use a module, you can use the import statement.

import math
print(math.sqrt(16))
Classes

Classes are a fundamental part of object-oriented programming (OOP). They define the blueprint for creating objects.

Example in Python:

class Dog:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def bark(self):
        return "Woof!"
Exceptions

Exceptions are errors detected during execution. They can be handled using try and except blocks.

try:
    result = 10 / 0
except ZeroDivisionError:
    print("Cannot divide by zero!")
